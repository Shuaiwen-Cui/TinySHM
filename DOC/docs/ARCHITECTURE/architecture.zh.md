# 架构

## 分层架构

```txt
+------------------------------+
| AI                           | <-- 基于低级函数的边缘设备 AI/ML 函数
+------------------------------+
| DSP                          | <-- 数字信号处理函数
+------------------------------+
| Math Operations              | <-- 各种应用的常用数学函数
+------------------------------+
| Adaptation Layer             | <-- 用平台优化/特定函数替换标准 C 中的函数
+------------------------------+
```

## 代码组织

```txt

+------------------------------+
| 应用层                        |
|   - measurement              | <-- 测量
|   - system_identification    | <-- 系统识别
|   - damage_detection         | <-- 损伤检测,定位和评估
+------------------------------+
| 中间件                        |
|   - TinyAdapter              | <-- 平台底层优化
|   - TinyMath                 | <-- 常用数学函数
|   - TinyDSP                  | <-- DSP 函数
|   - TinyAI                   | <-- AI 函数
+------------------------------+
| 驱动层                        |
+------------------------------+
| 硬件层                        |
+------------------------------+

```
